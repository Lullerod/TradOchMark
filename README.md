# TradOchMark
Förslag till ny hemsida för Träd och Mark i Gävle AB.

Sidan är byggd för statisk webbserver och innehåller 5 sektioner:
- Hero-section
- Tjänster med 4 klickbara boxar för att läsa mer
- Varför oss
- Om oss
- Submit form för att få kontakt.

Sidan är byggd med fokus på SEO optimering för att synas bättre utav sökmotorer och med typiskt call-to-action layout
för högre konvertering.
